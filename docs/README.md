# Launchpad AI

## Project Overview
This project aims to create an AI engine that serves as a foundation for various AI applications.

### Objectives
- Develop an AI engine with essential tools and functions.
- Use the AI engine to develop the Step Chef application that assists users in cooking by reformulating recipes and providing step-by-step guidance.

### Features
- Upload a picture of a recipe.
- Reformulate the recipe for maximum flavor.
- Step-by-step instructions with follow-up questions.
- Warnings and tips for chefs during the cooking process.

### Getting Started
1. Clone the repository from GitHub.
2. Install the required dependencies listed in `requirements.txt`.
3. Run the application and follow the instructions.
